Разработка программы "Учет книг в библиотеке" с использованием MySQL, C# и WPF
Требования к базе данных:
В базе данных должна быть таблица Books со следующими полями:
Id (INT, первичный ключ, автоинкремент)
Title (VARCHAR(255), название книги)
AuthorID (INT, внешний ключ, идентификатор автора книги)
YearPublished (INT, год публикации)
Genre (VARCHAR(100), жанр книги)
IsAvailable (BOOLEAN, доступность книги: true — книга доступна, false — книга выдана).
В базе данных должна быть таблица Authors со следующими полями:
Id (INT, первичный ключ, автоинкремент)
FirstName (VARCHAR(255), имя автора)
Patronymic (VARCHAR(255), отчество автора)
LastName (VARCHAR(255), фамилия автора)
Birthday (Date, дата рождения автора)

Требования к интерфейсу программы:
Главное окно :
Список всех книг из таблицы Books, включая информацию об авторах (использовать JOIN для получения данных).
Кнопки для выполнения операций: "Добавить", "Редактировать", "Удалить".
Поле поиска, позволяющее фильтровать книги по названию или автору.

Окно добавления/редактирования книги :
Поля для ввода данных: название,доступность,автор, год публикации, жанр, 
Кнопка "Сохранить" для подтверждения операции.

Окно добавления автора :
Поля для ввода данных: фио автора, год рождения.
Кнопка "Сохранить" для подтверждения операции.

Валидация данных:
Все поля обязательны для заполнения.
Год публикации должен быть числом и не может быть больше текущего года.
Название и автор не могут превышать 255 символов.
Год рождения автора должен быть меньше текущего года.

Подтверждение удаления :
При попытке удалить книгу должно появляться диалоговое окно с запросом подтверждения.

Требования к функционалу:
Просмотр книг :
Данные о книгах загружаются из базы данных при запуске программы.
Пользователь может обновить список книг, нажав кнопку "Обновить".
Добавление книги :
Пользователь может добавить новую книгу через форму добавления.
После добавления книги список обновляется автоматически.
Редактирование книги :
Пользователь может выбрать книгу из списка и отредактировать её данные.
Изменения сохраняются в базе данных.
Удаление книги :
Пользователь может удалить выбранную книгу из списка.
Удаление также удаляет запись из базы данных.
Добавление автора :
Пользователь может добавить нового автора через отдельную форму.
После добавления автор становится доступным для выбора при добавлении или редактировании книг.
Поиск книг :
Реализовать поиск по названию или автору (чувствительность к регистру не требуется).
Работа с авторами :
Пользователь может просматривать список всех авторов.
Возможность редактировать или удалять автора (при условии, что он не связан с книгами).

Требования к реализации:
Для работы с базой данных использовать библиотеку MySqlConnector
Программа должна корректно обрабатывать ошибки подключения к базе данных и другие возможные исключения (например, дублирование данных, некорректный ввод).
