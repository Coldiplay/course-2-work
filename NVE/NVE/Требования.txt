


Функциональные требования:
Авторизация:
Гость должен авторизоваться перед использованием приложения.

Просмотр учёта оборудования/инвентаря(нормально сформулировать)

Просмотр всех сотрудников:
Пользователь может просматривать информацию о сотрудниках(ФИО, должность, стаж, оклад).

Просмотр финансовых операций:
Автоматически идёт подсчёт всех зачислений и расходов за месяц, год.

Просмотр документов

Сортировка:
Пользователь может отсортировать финансовые операции по новизне, типу операции, сумме операции, сотрудников по ФИО, должности, стажу, окладу, оборудование по идентификационному коду, типу, количеству, названию.
Поиск:
Пользователь может найти финансовую операцию/и по дате операции, названию, описанию, сумме, сотрудника/ов по ФИО, должности, стажу, окладу, оборудование по идентификационному коду, типу, количеству, названию.
Фильтр:
Пользователь может отфильтровать операции по дате, типу операции, минимальной и максимальной сумме операции, сотрудников по ФИО, должности, стажу, минимальному и максимальному окладу, оборудование по идентификационному коду, типу, названию, количеству.

Добавление финансовой операции:
Пользователь может добавить финансовую операцию через отдельную форму.
Пользователь может настроить автоматическое создание финансовой операции через определённый промежуток времени.


Экспорт:
Пользователь может экспортировать финансовую операцию в документ.
