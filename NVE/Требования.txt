

Функциональные требования:

Пользователь может посмотреть статистику доходов и расходов за месяц, квартал, год.

Сортировка:
Пользователь может отсортировать финансовые операции по новизне, типу операции, сумме операции, категории операции, счёт по банку, балансу, валюте, вклад по валюте, банку, сумме вклада, дате открытия или закрытия, капитализации процентов, процентной ставке, периодичности выплат, типу вклада, долг по сумме, годовому %, дате взятия или конца выплаты долга.

Поиск:
Пользователь может найти финансовую операцию/и по дате операции, названию, описанию, сумме, вклад/ы по названию, сумме вклада, дате открытия или закрытия, долг/и по названию, сумме, дате взятия или конца выплаты долга, годовому %, счёт по названию, балансу, банку.

Фильтр:
Пользователь может отфильтровать операции по дате операции от xx.xx.xxxx и до xx.xx.xxxx, типу операции, минимальной и максимальной сумме операции, вклады по банку, валюте, минимальной и максимальной сумме вклада, датам открытия и закрытия вклада, периодичности выплат, капитализации процентов, типу вклада, долг по валюте, минимальной и максимальной сумме долга, датам взятия и закрытия долга, годовому %, счета по валюте, банку.

Пользователь может добавить финансовую операцию, категорию операций, долг, вклад, банк, счёт, валюту через отдельную форму.
Система автоматически подсчитывает статистику всех зачислений и расходов за месяц, квартал, год.
Система автоматически высчитывает подоходный налог.
Пользователь может настроить автоматическое создание финансовой операции через определённый промежуток времени.
Пользователь может переводить сумму денег между своими счетами.
Пользователь может покрыть часть долга из существующего счёта.
//Пользователь может пополнить вклад из существующего счёта.
