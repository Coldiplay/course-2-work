


Функциональные требования:
Авторизация:


Просмотр финансовых операций:
Автоматически идёт подсчёт всех зачислений и расходов за месяц, квартал, год.
Автоматическое высчитывание подоходного налога.

Сортировка:
Пользователь может отсортировать финансовые операции по новизне, типу операции, сумме операции
Поиск:
Пользователь может найти финансовую операцию/и по дате операции, названию, описанию, сумме
Фильтр:
Пользователь может отфильтровать операции по дате, типу операции, минимальной и максимальной сумме операции

Добавление финансовой операции:
Пользователь может добавить финансовую операцию через отдельную форму.
Пользователь может настроить автоматическое создание финансовой операции через определённый промежуток времени.
